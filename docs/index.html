🚀 About OOMlet

OOMlet is a lightweight, chaos-friendly QA and debugging Spring Boot application designed to help you test, stress, and harden your systems.

🧪 Testing Features

  • Dynamic health check toggling
  • Memory and file handle stress testing
  • CPU load simulation
  • Response latency simulation
  • Application crash simulation

🔧 Development Tools

  • Runtime log level adjustment
  • Graceful OS signal handling
  • Outbound connectivity testing
  • Full Actuator integration
  • Docker & Kubernetes support

📊 Quality Assurance

  • 80%+ code coverage enforcement
  • Comprehensive test suite
  • Security scanning integration
  • Automated CI/CD pipelines

🚀 Getting Started

Ready to start testing with OOMlet? Here's how to get started:

Quick Start

# Clone the repository
git clone https://github.com/trcjr/oomlet.git
cd oomlet

# Build the application
./mvnw clean package

# Run the application
java -jar target/oomlet-0.0.8.jar

Docker Quick Start

# Build and run with Docker
docker build -t oomlet:latest .
docker run -p 8080:8080 oomlet:latest

Kubernetes with Kind

# One-button setup
./scripts/setup-kind.sh

# Test the deployment
curl http://oomlet.local/actuator/health
